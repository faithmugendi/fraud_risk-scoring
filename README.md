# Fraud Detection and Risk Analytics Project

# Overview
This project demonstrates a data-driven approach to fraud detection using Python for data processing and Tableau for visualization. The goal is to identify high-risk financial transactions and support fraud detection and risk management efforts.

# Problem Statement
Financial fraud poses significant risks to financial institutions and public-sector programs. Detecting anomalous transactions is critical for maintaining financial integrity and preventing losses.

## Methodology
- Loaded and processed transaction data using Python (pandas)
- Developed a rule-based risk classification model (Low, Medium, High)
- Identified anomalous transactions using threshold-based analysis
- Generated a structured dataset for visualization

## Tools and Technologies
- Python (pandas)
- Tableau
- CSV data processing

## Key Features
- Risk level classification of transactions
- Identification of high-risk and anomalous transactions
- Data preprocessing for analytics workflows
- Integration with Tableau for visualization

## Results
The model successfully categorizes transactions based on risk levels and highlights potential anomalies, providing insights into fraud patterns and risk indicators.

## Project Structure
- fraud_data.csv (input dataset)
- fraud_output.csv (processed dataset)
- fraud_analysis.py (Python script)

## Relevance
This project reflects the application of data-driven analytics techniques to support fraud detection, financial oversight, and risk management systems.
